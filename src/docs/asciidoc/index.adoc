== Описание HTTP-сервиса работы с задачами
Настоящий документ содержит __вставил слова здесь на локалке Я__ техническое описание HTTP-сервиса работы с задачами (таск-трекера), структура данных которых приведена на xref:db/db.adoc[странице].

Данный сервис позволяет:

. Создавать задачи (xref:post/post.adoc[POST]).
. Получать список созданных задач (xref:get-all/get-all.adoc[GET]).
. Получать конкретную задачу из списка (xref:get/get.adoc[GET]).
. Изменять параметры задачи (xref:put/put.adoc[PUT]).
. Удалять задачу (DELETE).
