== Диаграмма последовательности "Оплата товара в интернет-магазине с применением промокода"

В данной диаграмме использован следующий функционал PlantUML:

* автонумерация действий
* алиасы для объектов-участников
* спец. примитивы объектов (actor, database и т.п.)
* расцветка примитивов объектов
* title
* skinparams
* цвет области участника
* строковые переменные
* комментарии
* activate/deactivate
* группировка участников в box (с выделением цветом).

Примечание: не использованы операторы alt, else.

[plantuml, babok.sequence, png]
----
include::babok.sequence.puml[]
